# Deepfake Gate — x402-Gated AI-Content Detection

Package: `@parad0x_labs/deepfake-gate` v0.1.0
Source: `packages/deepfake-gate/src/index.ts`

---

## Why This Exists

### EU AI Act — August 2026 Deadline

The EU AI Act (Regulation 2024/1689) enters full enforcement in August 2026.
Articles 50(2) and 50(4) require that:

- Providers of AI systems that generate or manipulate synthetic audio, image,
  or video content must mark that content in a machine-readable format.
- Deployers of those systems must ensure the synthetic origin is disclosed to
  end users.

Non-compliance carries fines up to EUR 15 million or 3% of global annual
turnover — whichever is higher.

Detection APIs (SynthID, BitMind, Hive) let platforms verify whether content
they receive is AI-generated before displaying it. These APIs cost money.
Deepfake Gate puts an x402 micropayment wall in front of each detection call
so the protocol earns USDC on every compliance check.

## Revenue Model

Every detection call routes through the DNA x402 facilitator:

```
Caller  →  x402 USDC payment  →  Facilitator  →  Detection API
                                      ↓
                             Protocol fee (10 bps)
                             + Provider cost pass-through
```

| Provider  | Caller pays (USDC) | Who gets what                    |
|-----------|-------------------|----------------------------------|
| SynthID   | 0.001             | Protocol fee + Google API cost   |
| BitMind   | 0.005             | Protocol fee + Bittensor routing |
| Hive      | 0.010             | Protocol fee + Hive API cost     |

At 10,000 detection calls/day (conservative for a mid-size social platform)
running the Hive provider, the protocol earns ~$100/day from compliance
traffic alone — traffic that must happen regardless of whether DNA x402
is the rail.

Callers never hold upstream API keys. They hold USDC.

## Providers

### Google SynthID (`synthid`)

- Detection method: **watermark**
- SynthID embeds an imperceptible signal in pixels/audio at generation time
  (Imagen 3, VideoPoet, Lyria).
- Best for detecting content that originated from Google infrastructure.
- Price: **$0.001 / call** — fastest, lowest cost.

### BitMind (`bitmind`)

- Detection method: **classifier**
- BitMind operates Bittensor subnet 34 — a decentralised network of
  classification models competing for accuracy.
- Independent of any single provider; handles GAN, diffusion, and hybrid
  content.
- Price: **$0.005 / call**

### Hive Moderation (`hive`)

- Detection method: **hybrid** (classifier + metadata forensics)
- Hive offers the broadest recall across generators (DALL-E, Midjourney,
  Stable Diffusion, Sora, and others).
- Recommended for platforms where content origin is unknown.
- Price: **$0.010 / call**

## NullLive Dual-Layer Attestation

NullLive (`@parad0x_labs/nulllive-sdk`) provides hardware-level proof that a
live stream was captured by a real physical camera, not generated by AI.
Attestation packets are anchored as Merkle roots on Solana every 30 seconds.

Deepfake Gate adds a second layer: even after a stream ends and is replayed as
a video file, the AI detector can verify no post-processing swapped the content.

```
buildNullLiveDetectionBadge(detectionResult, nullLiveBadge)
→ NullLiveDetectionBadge
```

Badge hierarchy:

| Level          | Condition                                        | Trust       |
|----------------|--------------------------------------------------|-------------|
| `hardware`     | NullLive verified (TeeCamera+) AND not AI        | Highest     |
| `ai-detector`  | AI detector: authentic, confidence >= 70%        | EU-compliant|
| `unverified`   | AI-generated OR low confidence OR no attestation | Flagged     |

## Usage

```typescript
import {
  DetectionProvider,
  detect,
  buildDetectionRequest,
  buildNullLiveDetectionBadge,
  DETECTION_PRICES_USDC,
} from "@parad0x_labs/deepfake-gate";

// Preview cost and endpoint before paying.
const preview = buildDetectionRequest(
  videoBytes,
  DetectionProvider.HIVE,
  "https://facilitator.dna-x402.xyz",
);
console.log(preview.estimatedPriceUsdc); // 0.01

// Run detection with x402 payment.
const result = await detect(videoBytes, DetectionProvider.HIVE, {
  facilitatorUrl: "https://facilitator.dna-x402.xyz",
  payerPublicKey: "YourBase58PubkeyHere",
  sign: async (challenge) => yourWallet.sign(challenge),
  anchorReceipt: true,   // anchor result hash on Solana
});

console.log(result.isAiGenerated); // false
console.log(result.confidence);    // 0-100
console.log(result.receiptHash);   // Solana tx signature

// Combine with NullLive hardware attestation.
const badge = buildNullLiveDetectionBadge(result, {
  status:   "verified",  // from nulllive-sdk verifyBadge()
  level:    2,           // TeeCamera
  anchorTx: "SolanaAnchorTxHere",
});

console.log(badge.level);     // "hardware"
console.log(badge.badgeText); // "Hardware-Verified Live Content"
```

## Compliance Notes

- Detection results with `anchorReceipt: true` are anchored on Solana and
  can serve as audit evidence under EU AI Act Article 50 record-keeping
  requirements.
- The `receiptHash` (Solana tx signature) can be included in content metadata
  (EXIF, C2PA manifest, or platform database) as proof of compliance check.
- Parad0x Labs does not store media content — only the SHA-256 hash is sent
  to detection providers.
