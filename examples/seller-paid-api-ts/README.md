# Seller Paid API Example

Private-pilot TypeScript seller API using the DNA seller SDK.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Test

```bash
npm test
```

## Expected Output

```txt
seller-paid-api: listening on http://127.0.0.1:3101
seller-paid-api: protected route /api/summary
```

The example uses manual/private-pilot settlement. Do not put production wallet keys in this example.
