# Copied Lot Ledger Example

Private pilot example for copied-lot entry, one-time finalization, and alpha fee accrual.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Expected Output

```txt
copied-lot-ledger example ok
status=CLOSED_WIN
alphaFee=1000
secondFinalize=REJECTED
```

## Acceptance

```bash
npm test
```
