# Agent Wallet Client Example

Private pilot example for registering a client-side generated agent wallet public key.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Expected Output

```txt
agent-wallet-client example ok
backendHasPrivateKey=false
```

This example does not generate or send a private key. Real clients must generate and encrypt agent wallet keys locally.

## Acceptance

```bash
npm test
```
