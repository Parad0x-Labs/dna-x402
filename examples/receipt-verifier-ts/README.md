# Receipt Verifier Example

Private-pilot receipt verifier example.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Test

```bash
npm test
```

## Expected Output

```txt
receipt-verifier: receipt verification helper ready
receipt-verifier: fee waterfall hash must match quoted waterfall
```

This example verifies signatures locally. It does not approve settlement by itself.
