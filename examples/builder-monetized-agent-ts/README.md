# Builder Monetized Agent Example

Private-pilot builder monetization example.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Test

```bash
npm test
```

```txt

builder-agent: DNA platform fee protected
builder-agent: builder fee visible
```
