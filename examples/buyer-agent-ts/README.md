# Buyer Agent Example

Private-pilot TypeScript buyer agent for DNA x402.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Test

```bash
npm test
```

## Expected Output

```txt
buyer-agent: quote request prepared
buyer-agent: commit payload prepared
buyer-agent: manual wallet signing required before finalize
```

This example does not move live funds. It prepares the buyer-side quote and commit loop.
