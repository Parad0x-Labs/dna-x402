Dark Passport — PRF → Ed25519 hardware-bound · no MPC · no relay

The first Solana-native hardware-bound passkey without MPC.
How it works: Your device's hardware authenticator runs a PRF (Pseudo-Random Function) keyed to a fixed salt. The 32-byte output is passed through HKDF-SHA-256 to produce an Ed25519 seed. Keypair.fromSeed(seed) yields a native Solana keypair. The private key is never stored anywhere — biometric authentication re-derives it on demand. Same device + same salt = same key, always.

Requires: Chrome 132+, Safari 18+, or Android Chrome (Pixel 6+).
