Browser Extension Installation Guide
Chrome
Edge
1

📂 Locate Extension Files

Open your file explorer and navigate to the extension folder:

pdx_dark_protocol/extension/

Make sure you can see files like manifest.json and popup.html

2

🔧 Open Chrome Extensions

Type this address in your Chrome address bar:

chrome://extensions/

Or click the 3 dots (⋮) → More tools → Extensions

3

⚙️ Enable Developer Mode

Look for the toggle switch in the top right corner labeled "Developer mode" and turn it ON.

Why? Chrome requires Developer Mode for extensions loaded from files (not from Chrome Web Store) as a security measure. This is normal and safe!
4

📦 Load the Extension

Click the "Load unpacked" button that appears after enabling developer mode.

Navigate to and select the pdx_dark_protocol/extension/ folder.

Click "Select Folder"

5

✅ Verify Installation

The PDX Dark extension should now appear in your extensions list.

Look for the extension icon in your Chrome toolbar (might be hidden behind the puzzle piece 🧩).

Click the icon to open your privacy wallet!

🚨 Having Trouble?

  • Extension not loading? Make sure you selected the correct folder containing manifest.json
  • Icon not visible? Click the puzzle piece (🧩) in toolbar and pin PDX Dark
  • Permission errors? Allow internet access when prompted
  • Still stuck? Check that Chrome/Edge is updated to latest version
