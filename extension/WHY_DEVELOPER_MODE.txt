=====================================
WHY DO I NEED DEVELOPER MODE FOR CHROME?
=====================================

Q: Why does PDX Dark require Developer Mode in Chrome?

A: Because we're installing from FILES, not from the Chrome Web Store!

Here's why this is necessary and safe:

🔒 SECURITY FEATURE
Chrome requires Developer Mode for extensions loaded from local files
to prevent malware and malicious extensions from being installed easily.

📁 UNPACKED VS PACKED
- Chrome Web Store extensions: Come as .crx files (safe, verified)
- Our extension: Loaded from folder/files (unpacked, requires Developer Mode)

✅ THIS IS NORMAL
- Phantom wallet requires this for local installation
- MetaMask requires this for local installation
- ALL browser extensions require this when loaded from files

✅ COMPLETELY SAFE
- Developer Mode only allows YOU to install extensions
- You control what gets installed
- PDX Dark is your own code - no hidden malware

✅ EASY TO REVERSE
- Want to remove Developer Mode later? Just toggle it off
- Extensions you installed will keep working
- No harm done to your browser

So yes, it's because of "unformed/unpacked installation" - but this is
by design for security, and it's exactly how professional wallets work! 🚀
